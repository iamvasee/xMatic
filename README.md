## 🚀 Quick Start Guide

### 📥 **Step 1: Install Extension**
```bash
# Option A: Chrome Web Store (Recommended)
Visit: https://chromewebstore.google.com/detail/xmatic/jhgjeaklmjohgmnephiaeiefejdhfnml

# Option B: Manual Installation
1. Clone this repository
2. Open Chrome → chrome://extensions/
3. Enable "Developer mode"
4. Click "Load unpacked" → Select extension/ folder
```

### ⚙️ **Step 2: Configure AI**
1. **Choose Provider**: OpenAI GPT or Grok AI
2. **Add API Key**: Get from [OpenAI Platform](https://platform.openai.com/) or [xAI](https://x.ai/)
3. **Configure Settings**: Temperature, max tokens, content style

### 🎯 **Step 3: Start Creating**
1. Go to [Twitter](https://twitter.com) or [X](https://x.com)
2. Compose a tweet
3. Click the 🤖 **AI button**

### 🚀 **Step 4: Generate & Publish**
- Create multiple tweets or replies
- Preview and edit content
- Publish with **one click**
