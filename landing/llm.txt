# xMatic - AI-Powered Twitter Content Generator

## Project Overview
xMatic is a Chrome extension that provides AI-powered Twitter content generation using your own API keys. It's designed to help users create engaging, viral tweets while maintaining complete privacy and control over their AI interactions.

## Key Features

### Core Functionality
- **AI-Powered Tweet Generation**: Generate tweets using OpenAI GPT or Grok AI APIs
- **Floating Panel Interface**: Multi-tab interface integrated directly within Twitter
- **Smart Publish System**: One-click publishing with Twitter intents and interface injection
- **Multiple Content Styles**: 9 different content styles including Professional, Casual, Humorous, Analytical, Concise, Empathetic, Creative, Enthusiastic, and Sarcastic
- **AI Reply Generation**: Generate contextual, engaging replies to existing tweets
- **Your Own API Keys**: Complete privacy and cost control using your own OpenAI/Grok API keys

### Technical Features
- **AI Control**: Full temperature & max tokens control for fine-tuning responses
- **Multiple Tweet Generation**: Generate multiple original tweets from context in one API call
- **Draft Management**: Organize and manage your content before publishing
- **Style Customization**: Customize AI responses to match your brand voice

## Installation & Setup

### Step 1: Install Extension
- Add xMatic from Chrome Web Store or load manually in developer mode
- Compatible with all Chromium-based browsers (Chrome, Edge, Brave)

### Step 2: Configure AI
- Choose between OpenAI GPT or Grok AI (xAI)
- Add your own API key for personalized AI access
- Configure temperature and max tokens for response control
